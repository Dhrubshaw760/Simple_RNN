# Simple_RNN

A deployable streamlit app that analyzes the sentiment of the reviewers using a simple RNN model.

### Data Collection and preprocessing
USing the imdb dataset to get the reviews dataset and apply feature representation on the data to convert the text into vector.

### Model Prediction
Simple Recurrent Neural Network was used to analyze the sentiment of viewers using various data from imdb and train the model on tanh activation function to make it learn complex patterns and eventually make accurate predicitons on unseen data.
